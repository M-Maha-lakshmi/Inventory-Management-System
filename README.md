# Inventory-Management-System
An Inventory Management System is a software application used to track, manage, and control inventory items in an organization

## Features
- Login
- Register
- Password hashing (BCrypt)
- Session authentication
- Repo + Service architecture
- Category module
- Product module
- Inventory module
- Pagination

## Technology
- ASP.NET Core MVC
- Entity Framework Core
- MySQL

## Required Packages

| Package | Version | Purpose |
|---|---|---|
| `BCrypt.Net-Next` | 4.1.0 | Password hashing — never store plain text passwords |
| `Microsoft.EntityFrameworkCore` | 8.0.25 | ORM for database access |
| `MySql.EntityFrameworkCore` | 8.0.22 | MySQL provider for EF Core |

## How to Run
1. Clone the repository
2. Update connection string
3. Run the project
