# Inventory-Management-System
An Inventory Management System is a software application used to track, manage, and control inventory items in an organization

## Features
- Login
- Register
- Password hashing (BCrypt)
- Session authentication
- Repo + Service architecture
- Category module
- Product module
- Inventory module
- Pagination

## Technology
- ASP.NET Core MVC
- Entity Framework Core
- MySQL

## Package to Install
dotnet add package BCrypt.Net-Next --version 4.1.0
dotnet add package Microsoft.EntityFrameworkCore --version 8.0.25
dotnet add package MySql.EntityFrameworkCore --version 8.0.22

## How to Run
1. Clone the repository
2. Update connection string
3. Run the project
